== Installation ==

We recommend you search for Pinpoll in WP-Admin, where installation is a breeze.

To install Pinpoll manually:

1. Download and unzip pinpoll.zip
2. Upload the "pinpoll" folder to your "/wp-content/plugins/" directory.
3. In WordPress, activate the plug-in through the "Plugins" menu.

And now, create your first poll :)
